မြန်မာနိုင်ငံစာကြည့်တိုက်အသင်း (MLA) ၏ မြန်မာပညာရပ်ခေါင်းစဉ်များ (Myanmar Subject Headings) GitHub repository အတွက် သပ်ရပ်ပြီး ပြည့်စုံတဲ့ Readme ဖိုင်မူကြမ်းကို အောက်ပါအတိုင်း ရေးသားပေးလိုက်ပါတယ်။
Myanmar Subject Headings (MLA Standard)
မြန်မာနိုင်ငံစာကြည့်တိုက်အသင်း (Myanmar Library Association - MLA) မှ သတ်မှတ်ထားသော မြန်မာပညာရပ်ခေါင်းစဉ်များ နှင့် Dewey Decimal Classification (DDC) စနစ်ကို အခြေခံထားသည့် ရှာဖွေရေးစနစ် (Search System) အတွက် Repository ဖြစ်ပါသည်။
📌 အကျဉ်းချုပ်
ဤ Project သည် မြန်မာစာကြည့်တိုက်လောကတွင် ပညာရပ်ဆိုင်ရာခေါင်းစဉ်များကို စနစ်တကျ အမျိုးအစားခွဲခြားနိုင်ရန်နှင့် နည်းပညာအသုံးပြု၍ လွယ်ကူစွာ ရှာဖွေနိုင်ရန် ရည်ရွယ်ပါသည်။ လက်ရှိတွင် ပညာရပ်ခေါင်းစဉ်ပေါင်း ၅,၈၆၇ ကျော်ကို စုစည်းထားပါသည်။
✨ အဓိကလုပ်ဆောင်ချက်များ (Features)
 * Multi-language Search: မြန်မာစာ၊ အင်္ဂလိပ်စာ သို့မဟုတ် DDC နံပါတ်များဖြင့် စိတ်ကြိုက်ရှာဖွေနိုင်သည်။
 * DDC Class Filtering: ဘာသာရပ်နယ်ပယ်အလိုက် (ဥပမာ - Philosophy, Science, Literature) ခွဲခြားရှာဖွေနိုင်သည်။
 * Offline Ready: အင်တာနက်မရှိဘဲ အသုံးပြုနိုင်ရန် စီစဉ်ထားသည်။
 * MLA Standard: မြန်မာနိုင်ငံစာကြည့်တိုက်အသင်း၏ စံနှုန်းများနှင့်အညီ ရေးဆွဲထားသည်။
📂 ဘာသာရပ်ခွဲခြားမှု (DDC Classes)
စနစ်အတွင်း၌ အောက်ပါ Main Classes များအတိုင်း ခွဲခြားထားပါသည် -
 * 0xx - General Works
 * 1xx - Philosophy & Psychology
 * 2xx - Religion
 * 3xx - Social Sciences
 * 4xx - Language
 * 5xx - Science
 * 6xx - Technology
 * 7xx - Arts
 * 8xx - Literature
 * 9xx - History & Geography
